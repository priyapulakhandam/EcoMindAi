# 🌱 EcoMind AI
### Multi-Agent Sustainability Intelligence System Powered by AI

> **EcoMind AI** is an intelligent sustainability analytics platform that helps organizations monitor energy consumption, detect anomalies, identify root causes, predict cost savings, evaluate environmental impact, and generate AI-powered sustainability recommendations using a **multi-agent AI architecture**.

Built with **Python, Streamlit, Google Gemini AI, Pandas, NumPy, and Plotly**, EcoMind AI transforms raw energy data into actionable sustainability insights through specialized AI agents.

# 🏗️ Multi-Agent Architecture

EcoMind AI follows a modular multi-agent architecture where each agent specializes in a specific sustainability task.

```

                        │
                        ▼
               Context Analysis Agent
                        │
                        ▼
             Anomaly Detection Agent
                        │
                        ▼
             Root Cause Analysis Agent
                        │
                        ▼

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/priyapulakhandam/EcoMindAi.git
```

```bash
cd EcoMindAi
```

---

## Create Virtual Environment (Optional)

```bash
python -m venv venv
```

Activate it:

Windows

```bash
venv\Scripts\activate
```

Linux/Mac

```bash
source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Configure Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_GOOGLE_GEMINI_API_KEY
```

---

## Run the Application

```bash
streamlit run streamlit_app.py
```
